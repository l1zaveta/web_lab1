# Отчет по работе

Интернет-магазин украшений ручной работы.

##  Функционал
- Каталог товаров с карточками украшений
- Добавление/удаление товаров из корзины
- Изменение количества товаров
- Автоматический подсчет суммы
- Сохранение корзины в localStorage
- Оформление заказа через модальное окно
- Полностью адаптивный дизайн

## Использованные технологии
- **HTML5** - семантическая разметка
- **CSS3** - Flexbox, Grid, медиазапросы
- **JavaScript** - логика


##  Адаптивность
- Десктоп
- Планшет: 2-3 карточки в ряд
- Смартфон: 1 карточка в ряд
